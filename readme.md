# NumPy Basics Notebook

This repository contains a Jupyter Notebook covering **core NumPy concepts** with simple, beginner-friendly examples.  
The purpose of this notebook is to understand how NumPy arrays work and how they differ from Python lists.

## Contents Covered

### 1. Introduction to NumPy Arrays
- Creating NumPy arrays
- Difference between Python lists and NumPy arrays
- Performance comparison using `process_time`

### 2. 1D Array Operations
- Creating 1D arrays
- Checking array shape

### 3. 2D Arrays
- Creating 2D arrays
- Understanding rows, columns, and shapes

### 4. Array Initialization in NumPy
- Arrays of zeros 
(`np.zeros`)
- Arrays of ones (`np.ones`)
- Arrays with fixed values (`np.full`)
- Identity matrices (`np.eye`)
- Random arrays (`np.random.random`, `np.random.randint`)
- Evenly spaced values using `np.linspace`

### 5. Analyzing NumPy Arrays
- Shape of an array (`.shape`)
- Number of dimensions (`.ndim`)
- Number of elements (`.size`)

### 6. Mathematical Operations
- Element-wise addition
- Subtraction
- Multiplication
- Division between arrays

### 7. Array Manipulation
- Transpose of matrices
- Reshaping arrays using `.reshape()`
